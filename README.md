## 📖 About the Project

This is a fun and interactive **Rock-Paper-Scissors** game where the user competes against the computer. The computer generates a random choice each round and the winner is decided by the classic game rules. Scores are tracked across all rounds and a final overall winner is declared at the end of the session.

This is **Task 4** of the CodSoft Python Programming Internship.

---

## ✨ Features

| Feature           | Description                                               |
| ----------------- | --------------------------------------------------------- |
| 🪨📄✂️ Core Game  | Full Rock-Paper-Scissors gameplay                         |
| 🤖 Computer AI    | Computer picks randomly each round                        |
| 📊 Score Tracker  | Tracks User, Computer and Tie scores                      |
| 🔢 Flexible Input | Accepts number keys (1/2/3) or text (rock/paper/scissors) |
| 🏁 Final Summary  | Displays overall winner after all rounds                  |
| ❌ Validation     | Handles invalid or unrecognized inputs                    |
| 🔄 Play Again     | Option to continue or quit after each round               |

## ▶️ How to Run

### Prerequisites

- Python 3.x must be installed → [Download Python](https://www.python.org/downloads/)

### Steps

```bash
# Step 1: Clone the repository
git clone https://github.com/limaraniray/CODSOFT-Task4-RockPaperScissors.git

# Step 2: Navigate into the folder
cd CODSOFT-Task4-RockPaperScissors

# Step 3: Run the application
python task4_rock_paper_scissors.py
```
